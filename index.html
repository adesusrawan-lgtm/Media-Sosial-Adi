Microsoft Excel

Microsoft Excel adalah program aplikasi spreadsheet (lembar kerja) berbasis baris dan kolom yang dikembangkan oleh Microsoft. Aplikasi ini dirancang untuk mengatur, menghitung, menganalisis, dan memvisualisasikan berbagai jenis data numerik maupun teks secara efisien menggunakan rumus, fungsi, dan grafik..
